# Skill: Rapid Prototype

## When to use
User wants to create a new app, describes features or screens.

## Tools
design, layout, compose_form, compose_tracker, compose_interactive, compose_display, persist, deploy

## Steps
1. Analyze requirements from user description
2. Design the visual system
3. Scaffold navigation
4. Compose UI components
5. Wire up persistence
6. Deploy
