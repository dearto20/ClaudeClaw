# Skill: Visual Refresh

## When to use
User wants to change theme, colors, or visual style of an existing app.

## Tools
design, deploy

## Steps
1. Review current visual system
2. Design updated theme (colors, typography, spacing)
3. Apply changes across the app
4. Deploy
