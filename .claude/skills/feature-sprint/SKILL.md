# Skill: Feature Sprint

## When to use
User wants to add, extend, or include new functionality in an existing app.

## Tools
compose_form, compose_tracker, compose_interactive, compose_display, persist, deploy

## Steps
1. Understand the existing app structure
2. Design the new feature
3. Compose required UI components
4. Wire up persistence if needed
5. Deploy
